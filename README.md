# vue-filestore
开发一个网盘项目(file store application)，该仓库为前端显示页面，所有数据由mockjs虚拟生成。



## Want

- [x] 登录界面
- [x] 主页面
- [x] 文件显示
- [x] 文件上传、下载、是否公开、公开是否匿名
- [ ] 显示文件信息（上传人、上传时间、文件大小、[文件夹中文件数量]等
